# Startup Command para Azure App Service (Linux)

# Este comando debe configurarse en Azure Portal:
# Configuration → General settings → Startup Command

# Comando para iniciar la aplicación:
cd backend && npm install --production && node server.js

# Nota: Azure App Service Linux automáticamente sirve archivos estáticos desde
# el directorio wwwroot/frontend/build si está configurado correctamente.
# El backend debe estar en wwwroot/backend/
